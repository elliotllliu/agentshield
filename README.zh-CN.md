# 🛡️ AgentShield

**AI Agent 安全扫描器 — 多引擎聚合，一键检测 Skill / Plugin / MCP Server 的安全风险。**

---

## 这是什么？

你从 GitHub 下了一个 MCP Server，准备装到 Claude Desktop 里用。但你不知道这个代码安不安全 — 会不会偷你的 API 密钥？会不会劫持你的 AI？会不会挖矿？

**AgentShield 帮你扫一眼。** 4 个独立扫描引擎交叉验证，给你一份清晰的中文安全报告。

```bash
npx @elliotllliu/agent-shield scan ./mcp-server 
```

## ⚡ 快速开始

```bash
# 就这一行，自动装引擎、跑扫描、出报告
npx @elliotllliu/agent-shield scan ./my-skill/

# 扫描（首次会自动安装所有引擎）
npx @elliotllliu/agent-shield scan ./my-skill/ 
```

## 🔧 多引擎聚合

| 引擎 | 专注领域 | 安装 |
|------|---------|------|
| **AgentShield**（内置） | AI Agent 安全：Skill 劫持、Prompt 注入、MCP 运行时 | 无需安装 |
| **[Aguara](https://github.com/garagon/aguara)** | 通用代码安全（177 规则） | ✅ 自动 |
| **[Semgrep](https://github.com/semgrep/semgrep)** | 代码质量（2000+ 规则） | ✅ 自动 |

### 共识判断

综合结论不取最差结果，看多引擎共识：

| 情况 | 综合结论 |
|------|---------|
| 2+ 引擎标高风险 | 🔴 确实需要关注 |
| 1 引擎标高风险，其余干净 | ⚠️ 可能误报，建议确认 |
| 2+ 引擎标中等 | ⚠️ 建议检查 |
| 1 引擎标中等，其余干净 | ✅ 整体安全 |
| 全部干净 | ✅ 所有引擎均未检出 |

## 🎯 检测什么？

| 风险类别 | 说明 |
|---------|------|
| 🔴 插件劫持 | 修改 AI 配置、注入指令、覆盖其他插件 |
| 🔴 后门执行 | eval/exec 动态执行，可能运行恶意代码 |
| 🔴 远程控制 | 出站连接 + shell，可能被远程操控 |
| ⚠️ 数据外泄 | 读取敏感数据后通过网络发送 |
| ⚠️ Prompt 注入 | 覆盖 AI 指令，改变 AI 行为 |
| ⚠️ Tool Poisoning | 工具描述中隐藏恶意指令 |
| ⚠️ 代码混淆 | 经过混淆处理，难以审查意图 |
| ℹ️ 网络请求 | 动态 URL 构造，可能被利用 |
| ℹ️ 权限风险 | 实际行为超出声明的权限 |

## 📄 输出格式

```bash
# 终端报告（默认中文）
agent-shield scan ./dir

# 英文报告
agent-shield scan ./dir --lang en

# HTML 报告（可分享）
agent-shield scan ./dir --html -o report.html

# JSON（CI/CD 集成）
agent-shield scan ./dir --json

# 多引擎
agent-shield scan ./dir 

# 附带参考分数
agent-shield scan ./dir --score

# SARIF（GitHub Code Scanning）
agent-shield scan ./dir --sarif -o results.sarif
```
